# Screenshots

This folder holds application screenshots referenced from the root [`README.md`](../README.md) for portfolio/demo purposes. None have been captured yet — add them here as PNGs with these exact names, then embed them in the root README (e.g. `![Dashboard](screenshots/dashboard.png)`):

| Filename | Suggested content |
|---|---|
| `login.png` | Login page |
| `register.png` | Registration page |
| `dashboard.png` | Dashboard — KPI cards + charts |
| `tickets.png` | Ticket list with search/filter/sort |
| `ticket-details.png` | Ticket detail page — timeline, comments, assignment panel |
| `create-ticket.png` | Create ticket form |
| `reports.png` | Reports page (PDF/Excel export) |
| `notifications.png` | Notification Center dropdown |

## How to capture

1. Run the app locally (see the root README's "How to Run the Project" section).
2. Log in with accounts of different roles (Employee vs. Agent/Manager) to capture role-specific views where relevant (e.g. the Assignment Panel only appears for Agent+).
3. Keep a consistent browser window size (e.g. 1440×900) across screenshots for a uniform look.
4. Save as PNG directly into this folder using the filenames above.
